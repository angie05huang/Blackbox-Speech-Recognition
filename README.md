# BlackBox AI Platform

An end-to-end AI-assisted transcription and document retrieval platform designed to streamline internal research workflows.

## Features
- Audio transcription using WhisperX
- Speaker diarization
- File upload and management
- Authentication and project-based access control
- Conversational querying over uploaded documents
- Downloadable PDF, CSV, and DOCX outputs

## Tech Stack
- Flask
- SQLAlchemy
- MySQL
- WhisperX
- Chroma Vector Database
- React
- OpenAI Embeddings
